# Apathy Theme for Zed

A dark, low-contrast theme family with muted purple undertones designed for reduced eye strain during long coding sessions.

## Themes Included

- **Apathy** - The original dark theme with subtle purple undertones
- **Apathetic Ocean** - A cooler, ocean-inspired variant
- **Minted** - A refreshing mint-accented dark theme

## Installation

### From Zed Extensions

1. Open Zed
2. Open the command palette (`Cmd+Shift+P` on macOS)
3. Search for "Extensions: Install Extension"
4. Search for "Apathy Theme"
5. Click Install

### Manual Installation

1. Clone or download this repository
2. Copy the `zed/` folder contents to your Zed extensions directory:
   - macOS: `~/.config/zed/extensions/apathy-theme/`
3. Restart Zed
4. Select the theme from Settings â†’ Themes

## Theme Selection

After installation, you can switch between themes:

1. Open Settings (`Cmd+,`)
2. Go to Themes
3. Select one of:
   - Apathy
   - Apathetic Ocean
   - Minted
